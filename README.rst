Installation
=============

Install::

    curl https://raw.github.com/creationix/nvm/master/install.sh | sh
    source ~/.nvm/nvm.sh
    nvm install 0.10
    nvm use 0.10
    npm install grunt-cli -g

Install dependences::

    bower install


Developerment
=============

Run debug server::

    grunt server
